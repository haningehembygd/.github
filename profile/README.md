# Haninge Hembygdsgille

Välkommen till Haninge Hembygdsgilles officiella GitHub-organisation.

Här samlar vi föreningens digitala projekt, öppna data, dokumentation och tekniska lösningar som stödjer vår verksamhet och bidrar till att bevara, dokumentera och sprida kunskap om Haninges kulturhistoria.

## Organisationens syfte

GitHub används som en gemensam plattform för att:

- bevara föreningens publika dokument i ett långsiktigt digitalt arkiv,
- utveckla och dokumentera digitala projekt,
- publicera öppna data och API:er,
- underlätta samarbete kring tekniska lösningar,
- skapa en stabil grund för framtida digital utveckling.

## Våra projekt

Bland organisationens repositories finns bland annat:

- **Digitalt Arkiv** – styrelseprotokoll, årsmöteshandlingar, stadgar och andra publika dokument.
- **Events API** – öppna JSON-data över föreningens evenemang.
- **Haninge Förr** – utveckling av föreningens digitala historiska tjänster.
- **Öppna data** – geografisk och kulturhistorisk information som kan återanvändas i appar och andra projekt.
- **Tekniska verktyg** – kod, dokumentation och hjälpprogram för föreningens webbplats och digitala tjänster.

## Öppenhet och transparens

Haninge Hembygdsgille arbetar för att göra föreningens offentliga handlingar och digitala resurser lättillgängliga. De flesta repositories är därför publika och kan läsas av alla.

Respektive repository innehåller information om vilka licens- och användningsvillkor som gäller för dess innehåll.
